# Azguard Wallet — Brand Kit

This brand kit contains the official Azguard Wallet logo and usage guidelines. The logo is a compass/sunburst mark split into a light and dark half, symbolizing security and direction.

## Logo Files

| File | Description | Best Used On |
|---|---|---|
| `Azguard_Raw_White_round.png` | Raw black-on-white version, round crop | Light backgrounds, print, favicons |
| `Azguard_Blue_Gradient_Round.png` | **Primary logo** — blue gradient, round crop | Light backgrounds, app icons, social avatars |
| `Azguard_Blue_Gradient_Square.png` | Blue gradient, square crop | App icons, stores requiring square format |
| `Azguard_White_Gradient_Dark_BG_Square.png` | White/silver gradient on dark background | Dark UI, dark mode, dark marketing backgrounds |
| `Azguard_Raw_Deep_Blue_Square.png` | Solid navy blue mark, square crop | Light backgrounds, flat single-color brand use |
| `Azguard_Raw_White_Square.png` | Solid white mark on dark background, square crop | Dark UI, dark mode, dark marketing backgrounds |

## Primary Mark

The **Blue Gradient** version is the primary brand mark and should be the default choice across all product surfaces (web app, mobile app, marketing site) whenever a colored background is not required.

- Gradient direction: top (dark navy blue) → bottom (bright sky blue)
- Approximate colors: `#0A2A5E` (dark navy) → `#3B9CFF` (bright blue)

## Usage Guidelines

**Do:**
- Use the round crop for avatars, favicons, and social profile images.
- Use the square crop for app icons and platforms that require square assets.
- Use the white/dark-background version on dark UI surfaces or dark photography.
- Maintain clear space around the logo equal to at least the width of one "ray" of the sunburst.

**Don't:**
- Don't recolor the gradient outside the approved blue palette.
- Don't stretch, skew, or rotate the mark.
- Don't place the blue gradient version on busy or low-contrast backgrounds.
- Don't place the dark-background version on light backgrounds (contrast will be too low).

## File Formats

All assets are currently provided as PNG.
